# Face Recognition Student Attendance Management System

## Overview

A real-time face recognition attendance system developed using Python, OpenCV, TensorFlow, and InsightFace.

The system automatically:

- Detects student faces
- Recognizes registered students
- Marks attendance automatically
- Tracks entry and exit times
- Calculates total attendance duration
- Stores attendance records in SQLite database

---

## Technologies Used

- Python
- OpenCV
- TensorFlow
- MediaPipe
- SQLite
- Tkinter

---

## Features

- Real-time face detection
- Face recognition using embeddings
- Attendance tracking
- Entry/Exit monitoring
- Attendance duration calculation
- Attendance reports
- GUI interface
